# InternFlow Backend Docs

Docs are grouped by purpose:

| Folder | Content |
| --- | --- |
| `api/` | API contracts and endpoint notes |
| `db/` | Database schema, ERD redirect, production baseline notes |
| `deployment/` | Render/deploy/env/CORS guides |
| `features/` | Feature design and implementation notes |
| `maintenance/` | Build fixes, help, operational notes |

Important root file:

```text
INTERNFLOW_DATABASE_RULES.md
```

Keep it at project root so DB-change instructions remain easy for agents and developers to find.
